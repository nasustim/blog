<h2>やること</h2>

$ minil new Acme::Nasustim::HogeHoge
$ cd Acme-Nasustim-HogeHoge

// 依存パッケージのインストール
cpanfileにモジュールを追記
$ carton install

// モジュール本体のコードとテストを書く
$ carton exec -- prove -Ilib t</pre>
