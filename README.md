# blog.nasustim.com

https://blog.nasustim.com

## Setup

Developing this project requires [asdf](https://asdf-vm.com/)

Installing runtimes it requires

```bash
$ asdf install
```

Installing npm packages

```bash
$ npm install
```

## Usage

Start dev server

```bash
$ npm run
```

Format code style

```bash
$ npm run lint:format
$ npm run textlint:format
```
